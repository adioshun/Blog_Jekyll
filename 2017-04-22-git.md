# Branch 관리
## 새로운 branch만들기 

```bash

git branch 새브랜치

```

## 새로운 branch에서 작업하기

```bash

git checkout 새브랜치

```

## 새로운 branch에서 작업한 내용 push하기 

```bash

git push --set-upstream origin 새브랜치
git push origin 새브랜치

```

## 새로운 branch에서 작업을 끝내고 Merge하기
```bash

git push --set-upstream origin 새브랜치
git push origin 새브랜치

```
